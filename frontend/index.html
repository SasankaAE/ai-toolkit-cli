Install in seconds

One command. Works on Windows, macOS, and Linux. No config needed — just set your OpenRouter key and go.

$powershell -c "irm https://raw.githubusercontent.com/SasankaAE/ai-toolkit-cli/master/install.ps1 | iex"
1
Open PowerShell and paste the command above.
2
Get a free API key at openrouter.ai and set it: $env:OPENROUTER_API_KEY="sk-or-..."
3
Run: ai chat ask-cmd "What is Python"
$curl -fsSL https://raw.githubusercontent.com/SasankaAE/ai-toolkit-cli/master/install.sh | bash
1
Open Terminal and paste the command above.
2
Set your key: export OPENROUTER_API_KEY="sk-or-..."
3
Run: ai chat ask-cmd "What is Python"

// usage

Commands

Everything available from the terminal.

$ai chat ask-cmd "..."
Ask any question. Supports any OpenRouter model via --model. Typewriter output with animated borders.
$ai summarize file README.md
Summarize any text file. Reads the file and returns a concise AI-generated summary.
$ai summarize text "..."
Summarize a string of text directly from the command line. No file needed.
$ai code generate "..."
Generate code from a plain-language description. Use --lang python to specify language.
$ai --help
Show all available commands and options with the animated banner.
